# :bird: Flappy bird

Another legendary arcade game. Push "Fly" button to avoid obstacles. Try to stay alive as long as possible and earn scores.

![Gameplay](screen.jpg)

## ✨ Game Features

- **Native assets:** Thanks for assets to https://github.com/samuelcust/flappy-bird-assets :heart:
- **Score:** High scores are saved to browser local storage

## 🕹️ Controls

**Click** or **Tap** - fly
