# :snake: Snake

Another legendary arcade game. Eat apples and avoid shit. Like real life.

## ✨ Game Features

- **Unique assets:** Simple but original assets made by me in a MS Paint
- **Score:** High scores are saved to browser local storage
- **Apples & shits** Every 5th eaten apple spawns shit on a field making game more and more difficult

## 🕹️ Controls

**Arrows** - change direction
**Space** - speed up
