# :yellow_square: Tetris

Legendary game.

![Gameplay](screen.jpg)

## ✨ Game Features

- **Unique graphic assets:** Pics assets made by me in a MS Paint
- **Score:** High scores are saved to browser local storage
- **Classic rules:** 10 x 20 grid, standart figures set, no modern features
- **Difficulty levels:** first one is boring, last one is insane

## 🕹️ Controls

- **Left, Right** - Move
- **Up** - Rotate
- **Down** - Speed boost
